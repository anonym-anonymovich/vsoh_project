#  Система мониторинга и защиты IoTустройств от аномального сетевого поведения

## Описание проекта
Проект представляет собой систему мониторинга и защиты IoT-устройств от аномального сетевого поведения. Система анализирует сетевой трафик IoT-устройств в реальном времени, обнаруживает отклонения от нормального поведения и предотвращает кибератаки.

Основная цель проекта — создание средства для защиты IoT-устройств путем анализа сетевого трафика и автоматического реагирования на угрозы. Система обеспечивает непрерывный мониторинг сети, обнаружение аномалий с использованием статистических методов и предоставление интуитивно понятного веб-интерфейса для управления безопасностью.

## Ключевые функции
 Анализ трафика IoT-устройств
 
 Обнаружение аномалий
 
 Создание и обновление профилей нормального поведения для каждого устройства
 
 Сигнатурный анализ — обнаружение известных паттернов атак (DDoS, сканирование портов)
 
 Уведомления о событиях — информирование администратора об обнаруженных угрозах

## Сценарии использования
### 1. Базовый мониторинг сети
Запуск системы для анализа трафика в локальной сети
Просмотр дашборда с перечнем всех IoT-устройств
Мониторинг текущего сетевого трафика и состояния устройств
### 2. Обнаружение DDoS-атаки
Система автоматически обнаруживает резкий рост трафика от конкретного устройства
В веб-интерфейсе отображается уведомление о DDoS-атаке
Администратор получает детальную информацию о типе атаки и источнике
### 3. Обнаружение сканирования портов
Система выявляет попытки подключения устройства к множеству различных портов
Фиксируется аномальное поведение и генерируется предупреждение
В отчете указывается конкретное устройство и тип обнаруженного сканирования
### 4. Обнаружение несанкционированной передачи данных
Система определяет передачу данных на внешние адреса, не входящие в список доверенных
Обнаруживается аномальный объем исходящего трафика
Администратор получает уведомление о возможной утечке данных
